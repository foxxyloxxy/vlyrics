# vlyrics

### Script to pull vocaloid lyrics from the vocaloidlyrics fandom wiki and annotate/color code by part of speech

### color key:
- blue: noun/proper noun
- green: verb
- orange: adjective/adverb
- white: particle
- pink: pronoun
- teal: determiner
- red: interjection
- purple: conjunction
- yellow: prefix/suffix/unknown
